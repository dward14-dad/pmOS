2. **Portfolio play** — every feature shipped demonstrates AI PM capability to employers

1. Coaching quality you can measure (eval harness → 90%+ satisfaction)
2. Real-user readiness (auth + data isolation)
3. An experience worth returning to (voice + flow)

## Why This Order

Strategy first forces the right questions before committing to a feature. PRDs before code ensures alignment on what "done" looks like. The auth PRD is first because it unblocks everything else — no real users, no real feedback without it.
